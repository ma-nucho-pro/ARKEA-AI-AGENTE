ARKEA AI - PAQUETE FINAL CONSOLIDADO

Este paquete reemplaza las versiones anteriores. Úsalo como base final para construir el instalador de Windows.

Archivo de compilación:
  build_windows_installer.ps1

Salida esperada:
  release\Arkea-AI-Setup-0.9.0.exe
o el nombre configurado por electron-builder.

Notas importantes:
- Las APIs externas requieren claves del usuario en Ajustes.
- Ollama se descarga desde el botón "Descargar Ollama".
- Los modelos locales se descargan por ID desde la interfaz.
- Las funciones que dependen de servicios externos no pueden funcionar sin internet/API key.
- El agente local funciona con SQLite, workspace por carpeta, chats, skills, memoria, cámara/pantalla, modelos y visualizador HTML.

AUTO-INSTALACIÓN INTEGRADA EN EL EXE
- El builder descarga OllamaSetup.exe y lo incluye como recurso del instalador.
- Al abrir ARKEA AI por primera vez, el proceso Electron intenta detectar Ollama.
- Si no está instalado, ejecuta el instalador incluido de Ollama.
- Si Ollama ya existe, no lo reinstala: intenta iniciar `ollama serve`.
- En segundo plano prepara faster-whisper tiny y descarga el pack mínimo de modelos:
  * gemma3:270m para chat rápido
  * qwen2.5-coder:0.5b para código
  * moondream:latest para visión ligera
  * nomic-embed-text para memoria/embeddings
- Esto no congela la interfaz; el chat queda disponible mientras se prepara el entorno.
- La instalación de modelos requiere internet la primera vez. Luego funcionan local/offline.
