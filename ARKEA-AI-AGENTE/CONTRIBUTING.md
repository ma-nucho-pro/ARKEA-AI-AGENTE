# Contribuir a ARKEA AI

Gracias por colaborar con ARKEA AI.

1. Crea un fork.
2. Crea una rama para tu cambio.
3. No incluyas claves, bases de datos ni archivos personales.
4. Ejecuta `python VALIDAR_REPOSITORIO.py`.
5. Abre un pull request explicando el cambio y cómo se probó.

Al contribuir aceptas que tus aportes se distribuyan bajo AGPL-3.0-or-later y que se conserven los avisos de autoría y atribución.
